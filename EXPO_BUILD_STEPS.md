## 📱 现在开始构建（5步搞定）

---

### 步骤1️⃣: 安装EAS CLI

打开终端，运行：

```bash
npm install -g eas-cli
```

⏱️ 需要时间：1-2分钟

---

### 步骤2️⃣: 登录Expo账号

```bash
eas login
```

**如果你还没有账号：**
1. 访问：https://expo.dev/signup
2. 填写邮箱和密码注册（免费）
3. 验证邮箱
4. 回来运行 `eas login` 登录

**如果已有账号：**
- 输入邮箱和密码登录即可

---

### 步骤3️⃣: 进入项目目录

```bash
cd /Users/myc/CodeBuddy/20260209021715
```

---

### 步骤4️⃣: 配置项目（首次需要）

```bash
eas build:configure
```

- 如果询问"Would you like to create a project?"，选择 `Yes`
- 会自动创建项目并关联到你的账号

---

### 步骤5️⃣: 开始构建APK 🎯

```bash
eas build --platform android --profile preview
```

**构建过程：**
1. 📤 上传代码（1-2分钟）
2. 📦 安装依赖（3-5分钟）
3. 🔨 编译APK（5-10分钟）
4. ✅ 完成！

**总时间：约10-15分钟**

---

## 📥 下载APK（4种方式）

构建完成后，你可以用以下任一方式下载：

### 方式1: 命令行链接 ⭐
- 终端会显示下载链接
- 复制链接在浏览器打开
- 点击下载

### 方式2: 邮件通知
- 检查你的注册邮箱
- 找到Expo发送的邮件
- 点击邮件中的链接下载

### 方式3: Expo网站
- 访问：https://expo.dev
- 登录账号
- 找到 `bill-tracker-app` 项目
- 点击最新的构建
- 下载APK

### 方式4: 手机扫码（最方便）📱
- 用手机扫描终端显示的二维码
- 直接在手机上下载
- 点击安装

## 💡 常见问题

### Q: 构建失败怎么办？
**A:** 
1. 查看终端错误信息
2. 截图或复制错误
3. 告诉我，我帮你解决

常见错误：
- 依赖版本冲突 → 我帮你修改
- 配置问题 → 我帮你调整
- 网络问题 → 重试即可

### Q: 免费额度够用吗？
**A:** 完全够！免费版每月30次构建，个人使用绰绰有余。

### Q: 可以中途取消吗？
**A:** 可以！按 `Ctrl+C` 取消上传，或在网站上取消构建。

### Q: 构建要钱吗？
**A:** 前30次/月免费，超过才收费（$29/月）。

### Q: 下载的APK能直接安装吗？
**A:** 能！下载后传到手机，点击安装即可。

### Q: 需要签名吗？
**A:** 不需要！Expo会自动处理签名。
